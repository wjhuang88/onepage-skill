## Summary

- First public release of the **onepage** skill.
- Skill renamed from `executive-html-report` to `onepage`, restructured under `skills/onepage/` with progressive reference loading.
- Adds design methodology (principles, Do's and Don'ts, visual pattern selection, component guardrails, known limitations, iteration guide) to produce more consistent output.
- Includes release workflow, install guide, dual-language README, and MIT license.

## Notable Changes

- **Renamed** from `executive-html-report` to `onepage`. All internal references updated (`SKILL.md` name, `openai.yaml` `$onepage` trigger, `AGENTS.md` constraints).
- **Decision Priorities** in SKILL.md: explicit 5-level decision weight (page mode → chapter flow → visual pattern → color mapping → content fill) with priority-tagged workflow steps.
- **Design methodology** added: 6 design principles, 7 Do's + 7 Don'ts, 8-pattern selection table with when-to-use/when-not-to-use, component guardrails for 7 component types, 6 known limitations, 7-step iteration guide.
- **Progressive reference loading**: `design-system.md` (always, 165 lines) + `layout-reference.md` (when building, 87 lines) + `advanced-guide.md` (complex tasks, 73 lines), down from a single 315-line file loaded every time.
- **Release workflow**: `.github/workflows/release.yml` produces zip + tar.gz + tar.zst on `v*` tag push.
- **Install guide**: `INSTALL.md` with agent directory table, version detection, fresh install, update, verify, Windows PowerShell, and troubleshooting.
- **Dual-language README**: English primary (`README.md`) + Chinese (`README.zh-CN.md`).

## Upgrade Notes

- This is the initial release. No upgrade path needed.
- Install by downloading the release archive and extracting into your agent's skill directory.
